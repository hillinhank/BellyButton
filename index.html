Project Overview

This webpages purpose is to visually diplay the results from Biodiversity data gathered from test participants Belly Buttons. These interactive charts aim to display the results of our study in a palatable manner. Select a study participant via the dropdown menu for 'Test Subject ID No:', then select the appropriate ID.

The data is visualized in three sections:

'Top 10 Bacteria Cultures Found' 'Belly Button Washing Frequency' 'Bacteria Cultures per Sample'
